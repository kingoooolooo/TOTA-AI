# Ayan AI Platform (Chatter AI)

A premium, glassmorphism-styled AI workspace that integrates chat and coding capabilities into a seamless, high-tech interface. 

This platform connects to Xiaomi MiMo models (via an OpenAI-compatible endpoint) and features a completely uncensored, unrestricted system prompt environment to allow for absolute AI freedom.

### 🌐 [Live Demo: Access the site directly here!](https://totaai.netlify.app/)

## ✨ Features

- **Uncensored AI Engine:** Custom system prompt injection removes safety guardrails for unrestricted, helpful responses from the underlying MiMo model.
- **Split-Panel Coding Workspace:** A dedicated "AI Coding Agent" panel sits next to the chat, automatically extracting, rendering, and syntax-highlighting code blocks generated by the AI.
- **Cinematic & Immersive UI:**
  - High-end dark theme utilizing glassmorphism and neon accents (cyan/purple).
  - Looping ambient video integrated seamlessly into the application sidebar.
  - Custom, rotated image backgrounds applied directly to the chat viewing area.
- **Client-Side Persistence:** Securely stores API keys locally in your browser to avoid server-side logging.
- **Fast & Modern Stack:** Built with React, TypeScript, and Vite for lightning-fast performance.

## 🛠️ Technology Stack

- **Frontend Framework:** React 18 with TypeScript
- **Build Tool:** Vite
- **Styling:** Custom CSS (Glassmorphism, CSS Modules/Globals)
- **State Management:** Zustand (Stores for settings, sessions, UI)
- **Icons:** Lucide React
- **Animations:** Framer Motion

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- A Xiaomi MiMo API Key

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/ayan-ai-platform.git
   cd ayan-ai-platform
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173/`. 
5. Go to the **Settings** panel (or API Key section) to enter your MiMo API Key and configure your model.

## 📂 Project Structure

- `/src/components/chat` - Chat interface components, message lists, input bars, and custom backgrounds.
- `/src/components/coder` - The AI coding agent panel, code editor, and preview functionality.
- `/src/components/layout` - Main app shell, sidebar with video looping, and navigation.
- `/src/components/settings` - API key configuration and model selection.
- `/src/stores` - Zustand stores managing application state.
- `/public` - Custom media assets like `sidebar-video.mp4` and `chat-bg.webp`.
